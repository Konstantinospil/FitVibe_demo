## Criterion

Plan progress tracking: progress_percent calculated as (completed_count / session_count) \* 100; progress updates when sessions are completed.

**SMART Criteria Checklist**:

- **Specific**: Clear calculation formula and update trigger
- **Measurable**: Progress percentage calculated correctly
- **Achievable**: Standard percentage calculation
- **Relevant**: User needs to see plan completion status
- **Time-bound**: Updates when sessions completed (real-time)
