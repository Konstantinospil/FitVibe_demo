## Criterion

API latency p95 ≤300ms for all endpoints per PRD targets; slow endpoints identified and optimized.

**SMART Criteria Checklist**:

- **Specific**: Clear latency target (p95 ≤300ms) and optimization requirement
- **Measurable**: Latency measured, slow endpoints identified
- **Achievable**: Standard performance optimization approach
- **Relevant**: User experience and performance
- **Time-bound**: ≤300ms p95 latency

## Test Method

Performance tests measure latency and identify slow endpoints.
