## Criterion

Frontend performance metrics meet targets: LCP ≤2.5s, CLS ≤0.1, TTI ≤3.0s on mid-tier device, 4G connection.

**SMART Criteria Checklist**:

- **Specific**: Clear performance targets (LCP, CLS, TTI) and test conditions
- **Measurable**: Metrics meet targets on specified device/connection
- **Achievable**: Realistic performance targets
- **Relevant**: User experience and performance
- **Time-bound**: Per-metric targets

## Test Method

Performance tests measure Core Web Vitals on mid-tier device, 4G connection.

## Evidence Required

- Lighthouse reports
- Web Vitals metrics
