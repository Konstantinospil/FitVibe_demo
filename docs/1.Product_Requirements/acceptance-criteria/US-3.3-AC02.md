## Criterion

Like counts update in real-time within ≤500ms; like count displayed on feed items and session details.

**SMART Criteria Checklist**:

- **Specific**: Clear update timing and display locations
- **Measurable**: Update time ≤500ms, displayed on feed and details
- **Achievable**: Realistic real-time update target
- **Relevant**: User experience requirement
- **Time-bound**: Update time ≤500ms

## Test Method

Integration tests verify update timing and E2E tests verify UI display.

## Evidence Required

- Count update tests
- UI screenshots
- API response times
