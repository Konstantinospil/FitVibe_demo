## Criterion

Read-only mode can be toggled via admin endpoint or environment variable; mode change takes effect within ≤10s.

**SMART Criteria Checklist**:

- **Specific**: Clear toggle mechanism and activation timing
- **Measurable**: Toggle works, mode change ≤10s
- **Achievable**: Standard toggle approach
- **Relevant**: Maintenance mode usability
- **Time-bound**: ≤10s activation

## Test Method

Integration tests verify mode toggle and activation timing.
