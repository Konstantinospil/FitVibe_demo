## User Story

**As a** system administrator  
**I want** read-only mode for maintenance  
**So that** I can perform maintenance without data loss

## Description

Read-only mode implemented via system configuration. Read-only mode prevents all write operations (POST, PATCH, DELETE return 503). Read-only mode can be toggled via admin endpoint or environment variable; mode change takes effect within ≤10s.
