## User Story

**As a** coach  
**I want** to modify exercise parameters (weight, exercise substitution, intensity/pace) when assigning training units  
**So that** I can personalize workouts for each athlete's level and needs

**Story Points**: 5

## Description

Coaches need the ability to modify exercise parameters during assignment to customize training units for individual athletes. This includes adjusting weight, substituting exercises, and modifying intensity/pace. These modifications apply only to the assigned session, not the original training unit template.

## Technical Notes

- Weight modification: adjust weight for strength exercises per athlete
- Exercise substitution: replace exercises with alternatives (e.g., different equipment, difficulty level)
- Intensity/pace modification: adjust intensity or pace for endurance exercises
- Modification scope: changes apply only to assigned session, not training unit template
- Validation: ensure substituted exercises are valid and compatible
- Modification tracking: record all modifications with coach and timestamp
