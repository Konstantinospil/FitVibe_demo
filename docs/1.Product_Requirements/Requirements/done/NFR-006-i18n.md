## Executive Summary

This non-functional requirement defines i18n standards and constraints for the FitVibe platform.

Support multiple languages and locales to serve international users.

## Business Context

- **Business Objective**: Support multiple languages and locales to serve international users.
- **Success Criteria**: English and German translations available, language preference persists, and locale formatting works.
- **Priority**: Medium
- **Quality Gate**: SILVER
- **Owner**: ENG
- **Status**: Proposed
- **Target Users**: All users (i18n affects everyone)

## Acceptance Criteria

Each acceptance criterion must be met for this requirement to be considered complete.

### NFR-006-AC01

**Criterion**: Static keys exist for EN/DE; missing‑key linter passes CI with 0 missing keys.

- **Test Method**: Lint
- **Evidence Required**: Lint logs

### NFR-006-AC02

**Criterion**: Language toggle persists across sessions; locale formats match selection.

- **Test Method**: E2E
- **Evidence Required**: Screens

### NFR-006-AC03

**Criterion**: No hard-coded user-facing strings in covered paths.

- **Test Method**: Static
- **Evidence Required**: Lint logs

## Use Cases

### Primary Use Cases

- User selects language (EN/DE)
- User views UI in selected language
- User sees dates/numbers in locale format
- User language preference persists

### Edge Cases

- Translation key is missing (fallback to EN)
- User switches language mid-session
- Locale formatting fails for edge cases

## Dependencies

### Technical Dependencies

- i18n library
- Translation files (JSON)
- Locale formatting library

## Constraints

### Technical Constraints

- Static keys for EN/DE
- No hard-coded strings
- Missing key linter passes

### Business Constraints

- English and German supported
- More languages may be added later

## Assumptions

- Translations are accurate
- Users understand language selection
- Locale data is available

## Risks & Issues

- **Risk**: Missing translations may confuse users
- **Risk**: Locale formatting may be incorrect
- **Risk**: Translation maintenance may be overlooked
